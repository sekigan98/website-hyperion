Política de Privacidad

Última actualización: 28/12/2025

Esta Política describe qué datos recolecta Hyperion, para qué se usan y qué opciones tenés. Aplica al sitio web, el dashboard, la API y el cliente de escritorio.

1) Qué datos recolectamos

  • Cuenta: email y credenciales (la contraseña debe almacenarse hasheada en el servidor).
  • Sesión: token de autenticación (JWT o equivalente) para mantenerte logueado.
  • Licencias: estado de licencias, plan asociado, activaciones realizadas.
  • Activación del cliente: identificador de instalación (“installId”), nombre de dispositivo (si se envía), versión de la app.
  • Telemetría mínima (opcional): errores, logs técnicos y métricas básicas para mejorar estabilidad.
  • Datos técnicos: IP, user-agent y eventos de seguridad (por ejemplo, intentos de login fallidos).

2) Para qué usamos los datos

  • Crear tu cuenta, autenticarte y mantener tu sesión.
  • Validar licencias y aplicar límites del plan.
  • Prevenir fraude/abuso y mejorar seguridad.
  • Soporte técnico y mejora del producto (bugs, rendimiento).

3) Cookies y almacenamiento local

El sitio puede usar almacenamiento local del navegador (por ejemplo, localStorage) para guardar el token de sesión. Si en el futuro se agregan cookies, se usará lo mínimo necesario para autenticación/seguridad.

4) Compartir datos con terceros

Podemos usar proveedores para hosting, base de datos, email o analítica. Compartimos solo lo necesario para operar el servicio. No vendemos tus datos personales.

5) Seguridad

  • Usar HTTPS para el sitio y la API.
  • Hashear contraseñas (por ejemplo, bcrypt/argon2) y proteger tokens.
  • Aplicar controles de acceso y logs de seguridad.

6) Retención

Conservamos datos el tiempo necesario para prestar el servicio y cumplir obligaciones legales. Podés pedir eliminación/cierre de cuenta (ver Contacto).

7) Tus derechos

Podés solicitar acceso, rectificación o eliminación de tus datos, y pedir asistencia sobre tu cuenta. Escribinos a soporte indicando el email registrado.

8) Menores

Hyperion no está dirigido a menores. Si sos menor de edad, necesitás autorización de tu responsable legal.

9) Cambios

Podemos actualizar esta Política. Publicaremos la versión vigente en esta página.

10) Contacto

Soporte: soporte@hyperion.com
